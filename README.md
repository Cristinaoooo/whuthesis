whuthesis
=========

这是武汉大学本科毕业论的模板，整个模板分为3个部分：

1. whuthesis.cls, 这里定义文档类中的宏
2. whuthesis.cfg, 这里存放固定变量以及配置
3. test.tex, 测试模板
